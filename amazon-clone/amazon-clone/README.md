# Amazon E-commerce Clone 🛒

A full-stack Amazon-like e-commerce web application built using modern web technologies.

---

## 🚀 Features
- Product listing by category
- Product detail page
- Add to cart functionality
- Checkout process
- Order management system
- Responsive UI similar to Amazon

---

## 🛠️ Tech Stack
- **Frontend:** Next.js, React
- **Backend:** Node.js, Express
- **Database:** MySQL

---

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/Tamanakatoch-24/amazon-ecommerce-clone.git

2. Navigate to project folder:
   cd amazon-ecommerce-clone

3. Install dependencies:
   npm install

4. Run the project:
   npm run dev

---

## 📌 Assumptions
- No authentication (default user assumed logged in)
- Sample product data used for demonstration
